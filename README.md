# cse291b-covid19
Repository for CSE291B-Big Data and Knowledge Spring 2026.

## Project Description
This project is formulated as a supervised multi-class classification problem using high-dimensional proteomic measurements derived from the MassIVE dataset MSV000085507 (RMSV000000331.3). The dataset contains proteomic measurements from 90 patients distributed across four conditions: 22 Healthy, 25 Non-severe COVID-19, 18 Severe COVID-19, and 25 Symptomatic non-COVID-19.
